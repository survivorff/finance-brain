# 🟡 Level 2: 建立框架

> 目标：建立宏观经济分析框架，理解经济运行的底层逻辑，能独立判断经济周期位置。

## 完成标准

学完 Level 2 后，你应该能：
- [ ] 画出经济周期的四个阶段，并说出每个阶段的特征
- [ ] 解释央行加息/降息的传导机制
- [ ] 理解"信用创造"如何驱动经济扩张和收缩
- [ ] 判断当前中国和美国分别处于周期的哪个阶段
- [ ] 说出不同经济环境下应该配置什么资产
- [ ] 理解汇率变动对资产价格的影响链条

- 《货币金融学》— 米什金（教材，选读）
