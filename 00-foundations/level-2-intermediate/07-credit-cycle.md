# 07 信用与债务周期 | Credit & Debt Cycle

`🟡 进阶` `预计阅读：30 分钟`

> 核心问题：为什么债务是经济最大的驱动力，也是最大的风险源？什么是"资产负债表衰退"？中国会重蹈日本覆辙吗？

---

## 一句话总结

**经济的本质是信用扩张和收缩的循环。债务能创造繁荣，也能制造危机。理解债务周期，就理解了宏观经济的"主线"。**

---

## 信用为什么这么重要？

```mermaid
graph LR
    A[一个人的支出<br/>= 另一个人的收入]
    B[支出来源] --> C[自己赚的钱]
    B --> D[借来的钱<br/>= 信用]
    
    E[信用扩张] --> F[支出增加 → 收入增加 → 经济繁荣]
    G[信用收缩] --> H[支出减少 → 收入减少 → 经济衰退]
```

> 💡 达里奥的核心观点：**经济中绝大部分增长来自信用，而不是生产力**。生产力增长缓慢稳定，但信用可以快速扩张和收缩，这就是周期的本质。

## 短期债务周期（5-8 年）

```mermaid
graph TB
    A[复苏期] --> A1[利率低，借钱便宜<br/>信用扩张<br/>消费投资增加]
    A1 --> B[繁荣期]
    B --> B1[经济活跃<br/>资产价格上涨<br/>通胀抬头]
    B1 --> C[过热期]
    C --> C1[央行加息<br/>抑制过热<br/>信用扩张放缓]
    C1 --> D[衰退期]
    D --> D1[借款减少<br/>支出下降<br/>失业上升]
    D1 --> E[央行降息<br/>重新刺激信用]
    E --> A
```

这是常规的"美林时钟"周期，由央行通过利率工具调节。

---

## 长期债务周期（50-75 年）

```mermaid
timeline
    title 长期债务周期
    section 早期阶段
        债务/GDP 较低 : 健康借贷
        生产力增长 : 真实繁荣
    section 泡沫阶段
        债务快速积累 : 资产价格暴涨
        收入跟不上 : 越来越多投机
    section 顶部
        债务/GDP 极高 : 利率到顶
        加息刺破泡沫 : 资产价格暴跌
    section 去杠杆
        债务违约 : 资产价格崩盘
        央行降息救助 : 但难以扭转
    section 美丽去杠杆
        通缩+紧缩 : 调整缓慢痛苦
        通胀化解 : 债务实际贬值
```

> 💡 短周期是"利率周期"，长周期是"债务周期"。当长周期顶部到来时，短周期工具（降息）会失效——这就是当前世界面临的局面。

## 达里奥的"美丽去杠杆"框架

去杠杆有四种方式：

```mermaid
graph TB
    A[去杠杆方式] --> B[1. 紧缩支出<br/>痛苦]
    A --> C[2. 债务违约<br/>痛苦]
    A --> D[3. 财富转移<br/>政治阻力]
    A --> E[4. 央行印钱<br/>通胀化解]
    
    F[最理想：四者结合] --> G["美丽去杠杆"<br/>Beautiful Deleveraging]
    G --> H[通胀适度上升<br/>债务实际贬值<br/>但不至于失控]
```

### 历史上的去杠杆

| 时期 | 国家 | 方式 | 结果 |
|------|------|------|------|
| 1930s 大萧条 | 美国 | 紧缩+违约 | 灾难（GDP -25%） |
| 1990s | 日本 | 紧缩+缓慢印钱 | 失去三十年 |
| 2008 后 | 美国 | 大规模 QE + 财政 | 美丽去杠杆 |
| 2022+ | 美国 | 通胀化解 + 高名义增长 | 进行中 |
| 2021+ | 中国 | 多种方式同时 | 正在尝试 |

## 资产负债表衰退（辜朝明）

这是理解日本"失去三十年"和当前中国的关键概念。

```mermaid
graph TB
    A[资产价格暴跌] --> B[企业资产 < 负债<br/>= 技术性破产]
    B --> C[但仍有现金流<br/>不会立刻倒]
    C --> D[企业行为转变：<br/>不再是利润最大化<br/>而是负债最小化]
    D --> E[拼命用现金流还债<br/>不再借新债<br/>不再投资]
    E --> F[需求消失<br/>即使利率为零也无效]
    F --> G[长期通缩]
    
    style D fill:#e74c3c,color:#fff
```

### 资产负债表衰退 vs 普通衰退

| | 普通衰退 | 资产负债表衰退 |
|--|----------|---------------|
| 触发 | 加息过度/外部冲击 | 资产泡沫破裂 |
| 持续 | 1-2 年 | 10-20 年 |
| 利率政策 | 有效 | 失效 |
| 唯一对策 | 等周期 | 大规模财政 |
| 案例 | 2001 互联网泡沫后 | 日本 1990s |

## 信用周期的领先指标

```mermaid
graph TB
    A[领先指标<br/>提前 6-12 个月] --> B[信用脉冲<br/>Credit Impulse]
    A --> C[银行愿意贷款的程度<br/>SLOOS 调查]
    A --> D[信用利差<br/>高收益债 vs 国债]
    A --> E[杠杆率变化]
    
    F[同步指标] --> G[新增贷款]
    F --> H[企业债发行量]
    F --> I[房贷申请量]
    
    J[滞后指标] --> K[违约率]
    J --> L[坏账率]
```

### 信用脉冲 (Credit Impulse)

```
信用脉冲 = 新增信贷 / GDP 的二阶导数
       = 信用增速的"加速度"

> 0：信用扩张加速 → 经济会变好
< 0：信用扩张减速 → 经济会变差
```

> 📊 中国的信用脉冲领先 GDP 增速约 2-3 个季度。这是判断中国经济拐点的核心指标之一。

## 当前的核心矛盾

```mermaid
graph TB
    A[全球处于长期债务周期顶部] --> B[各国债务都接近极限]
    
    B --> C[加息会让付息成本爆炸<br/>→ 央行被迫"忍受"通胀]
    B --> D[降息会让通胀失控<br/>→ 央行不能轻易宽松]
    B --> E[财政受限<br/>→ 政策空间收窄]
    
    F[出路] --> G[1. 通胀化解<br/>金融抑制]
    F --> H[2. 经济增长摆脱<br/>需要技术革命/AI?]
    F --> I[3. 债务重组/违约<br/>痛苦]
    F --> J[4. 战争/危机<br/>不可控]
    
    style A fill:#e74c3c,color:#fff
```

> 💡 这是为什么近几年"金融抑制"(Financial Repression) 概念兴起——政府通过低利率 + 通胀来"温水煮青蛙"地降低实际债务。

## 核心概念速查

| 术语 | 英文 | 一句话解释 |
|------|------|-----------|
| 信用 | Credit | 借贷形成的债权债务关系 |
| 杠杆 | Leverage | 债务相对自有资本的倍数 |
| 去杠杆 | Deleveraging | 减少债务/降低杠杆 |
| 信用脉冲 | Credit Impulse | 信用扩张速度的变化 |
| 资产负债表衰退 | Balance Sheet Recession | 私人部门主动去杠杆 |
| 美丽去杠杆 | Beautiful Deleveraging | 多种工具组合温和去杠杆 |
| 金融抑制 | Financial Repression | 政府用低利率+通胀降低实际债务 |
| 债务货币化 | Debt Monetization | 央行直接为政府融资 |
| 主权债务危机 | Sovereign Debt Crisis | 政府还不起债 |
| 庞氏债务 | Ponzi Debt | 借新还旧维持的债务 |
