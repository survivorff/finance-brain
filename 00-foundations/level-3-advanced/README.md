# 🔴 Level 3: 独立分析

> 目标：能独立做投研分析，形成自己的市场判断，建立初步的投资体系。

## 前置要求

完成 [Level 2: 建立框架](../level-2-intermediate/) 全部内容。

## 完成标准

学完 Level 3 后，你应该能：
- [ ] 用 DCF 给一家公司估值
- [ ] 独立写一份行业研究报告
- [ ] 从宏观数据推导出资产配置建议
- [ ] 解读一次 FOMC 会议并判断市场影响
- [ ] 建立自己的投资检查清单 (Checklist)
- [ ] 写出完整的投资决策记录（为什么买/卖/不动）
