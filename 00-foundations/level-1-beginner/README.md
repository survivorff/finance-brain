## 完成标准

学完 Level 1 后，你应该能：
- [ ] 解释通胀是怎么产生的
- [ ] 说出央行加息/降息的基本逻辑
- [ ] 区分股票、债券、基金的本质区别
- [ ] 看懂一条财经新闻里的关键数据
- [ ] 理解"风险"不等于"亏钱"
