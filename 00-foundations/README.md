## 学习建议

1. **不要跳级**。每一层都是下一层的地基。
2. **边学边记**。用 [07-journal](../07-journal/) 记录你的理解和困惑。
3. **结合实事**。学完一个概念，去 [05-daily-tracking](../05-daily-tracking/) 找真实案例对照。
4. **反复回看**。随着认知提升，同一篇笔记会有不同的理解。
