## 一句话总结

**债券市场比股票市场大得多，是金融体系的"地基"。理解债券，就理解了利率，就理解了所有资产的定价基础。**

## 2022 年股债双杀：为什么？

```mermaid
graph TB
    A[2022 年特殊环境] --> B[通胀高企<br/>美 CPI 9%]
    A --> C[美联储暴力加息<br/>0% → 4.5%]
    
    B --> D[股票承压<br/>估值压缩]
    C --> D
    C --> E[债券暴跌<br/>新债收益率高，旧债贬值]
    
    D --> F[股债双杀<br/>60/40 组合最大回撤 ~20%]
    E --> F
    
    G[60/40 组合的脆弱前提：<br/>股债负相关] --> H[在通胀环境下失效<br/>都受加息冲击]
```

> 💡 这就是为什么有人开始讨论"60/30/10"（股/债/商品）的新均衡。

## 信用利差 (Credit Spread)

```
信用利差 = 信用债收益率 - 国债收益率（同期限）
```

```mermaid
graph TB
    A[信用利差扩大] --> B[市场对信用风险定价上升]
    B --> C[经济恶化预期 / 风险厌恶]
    
    D[信用利差收窄] --> E[市场风险偏好上升]
    E --> F[经济乐观 / 流动性宽松]
```

历史关键节点：
- 2008 金融危机：美国高收益债利差飙升至 20%+
- 2020 疫情：高收益债利差短期飙升至 11%
- 2022 加息：利差扩大但未到危机级
- 平时：高收益债利差 3-5%

## 债券投资策略

### 1. 久期管理

```mermaid
graph TB
    A[预期降息] --> B[拉长久期<br/>买长期债券]
    B --> C[利率↓时<br/>长债涨幅大]
    
    D[预期加息] --> E[缩短久期<br/>买短期债券或货基]
    E --> F[避免长债下跌]
```

### 2. 信用下沉

```mermaid
graph LR
    A[低风险偏好] --> B[只买国债/AAA]
    C[中风险] --> D[AA+/AA 城投]
    E[高风险偏好] --> F[高收益债 / 垃圾债]
```

### 3. 可转债策略

可转债 = 债券底（保本）+ 转股权（弹性）

```mermaid
graph TB
    A[股价低于转股价] --> B[当债券持有<br/>赚票息]
    C[股价上涨] --> D[转股<br/>赚股价上涨]
    
    E[特点：<br/>下有保底<br/>上不封顶]
```

## 核心概念速查

| 术语 | 英文 | 一句话解释 |
|------|------|-----------|
| 收益率 | Yield | 持有债券的年化回报 |
