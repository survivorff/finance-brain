| 房地产 | 土地 + 建筑 | 租金 + 增值 | 利率、政策 | 负相关 |

## 跨资产思维

学习单个资产是基础，但真正的能力是**理解资产之间的关联**：

- 美债收益率上升 → 股票估值承压 → 但银行股受益
- 美元走强 → 黄金承压 → 新兴市场资金外流
- 油价上涨 → 通胀预期上升 → 加息预期升温

详见 → [全球经济关联分析](../04-global-economy/connections/)

## 研究模板

每个资产的研究笔记建议包含：

1. **本质**：这个资产到底是什么？
2. **驱动因素**：什么决定它的价格？
3. **估值方法**：怎么判断贵不贵？
4. **风险特征**：最大的风险是什么？
5. **与其他资产的关系**：和谁正相关？和谁负相关？
6. **当前状态**：现在处于什么位置？
