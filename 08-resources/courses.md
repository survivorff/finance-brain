## 推荐学习路径

### Phase 1：基础（1-3 月）
1. ✅ Ray Dalio "How the Economic Machine Works"（30 分钟视频）
2. ✅ 薛兆丰经济学课（精选章节）
3. ✅ 银行螺丝钉指数基金入门
4. ✅ 阅读《漫步华尔街》

### Phase 2：进阶（3-6 月）
1. ✅ Robert Shiller 耶鲁金融市场（Coursera 免费）
2. ✅ Damodaran Valuation Course（YouTube）
3. ✅ 阅读《周期》《投资最重要的事》
4. ✅ 香帅金融学课程

### Phase 3：高级（6-12 月）
1. ✅ Damodaran Corporate Finance
2. ✅ MIT 15.401 Finance Theory
3. ✅ 阅读《债务危机》《聪明的投资者》
4. ✅ 跟踪深度 podcast：Macro Voices, Forward Guidance

### Phase 4：精通（持续）
1. ✅ 阅读《思考，快与慢》《反脆弱》
2. ✅ 关注顶级研究：Pozsar, Druckenmiller 访谈
3. ✅ 自己写研究、做决策、复盘
4. ✅ 与高质量同好持续交流

## 学习心法

### 1. 不要追求"完成所有课程"

```
深度学习 1 门好课 > 浅尝 10 门
重点是把核心概念真正理解
```

### 2. 边学边实践

```
学完 DCF → 给一家公司估值
学完美林时钟 → 判断当前阶段
学完资产配置 → 设计自己的组合
```

### 3. 多元化来源

```
中文 + 英文：视角互补
书 + 课 + 播客：媒介互补
理论 + 实战 + 复盘：层次互补
```

### 4. 持续 > 一阵风

```
每天 30-60 分钟 > 一周一次的"突击"
养成习惯比"刻苦学一阵"更有效
```

### 5. 保持批判

```
即使是大师的观点，也要用自己的逻辑验证
不要崇拜任何人
建立自己的思考框架
```

## 免费 vs 付费

```
免费资源已经足够你成为优秀投资者：
- Yale/MIT/NYU 公开课
- Damodaran YouTube
- 桥水公开报告
- 各国央行公开数据
- 经典书籍（图书馆都有）

付费的价值：
- 时间节省（精选+整理）
- 系统化（避免遗漏）
- 互动（社群+提问）
- 但不是必需的
```
