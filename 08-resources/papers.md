## 怎么读论文？

### 1. 不需要读完整篇

```
读论文的层次：
1. 标题 + 摘要（80% 论文止于此）
2. 引言 + 结论（再筛选 80%）
3. 数据和方法（精读重点论文）
4. 完整阅读（极少）

→ 学会快速筛选
```

### 2. 关注"反共识"论文

```
教科书内容已经熟悉了
寻找"挑战共识"的研究
往往启发更大
```

### 3. 注意时效性

```
30 年前的论文可能已经过时
近 5-10 年的研究更有参考价值
（除非是经典理论）
```

### 4. 横向对比

```
同一主题看 3-5 篇不同观点的论文
不要被单一研究牵着走
```

## 论文阅读清单（按主题）

### 想理解市场效率？

1. Fama (1970) - Efficient Capital Markets
2. Shiller (1981) - Volatility Tests
3. De Long (1990) - Noise Trader Risk

### 想理解危机？

1. Reinhart & Rogoff (2009) - This Time Is Different
2. Brunnermeier & Pedersen - Funding Liquidity Spirals
3. Krugman (1979) - Currency Crises

### 想理解资产配置？

1. Markowitz (1952) - Portfolio Selection
2. Sharpe (1964) - CAPM
3. Brinson 等 (1986) - Determinants of Portfolio Performance

### 想理解中国经济？

1. Michael Pettis 的研究（必看）
2. 央行工作论文
3. NBER 关于中国的论文
