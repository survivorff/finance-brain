## 仓位管理

### 凯利公式（简化版）

```
最优仓位 = 胜率 - (1 - 胜率) / 赔率

例：胜率 60%，盈亏比 2:1
最优仓位 = 0.6 - 0.4/2 = 40%
```

> ⚠️ 实战中通常用凯利公式的 1/4 或 1/2，因为我们对胜率和赔率的估计不准。

### 单一仓位上限

| 风险偏好 | 单一资产上限 | 单一行业上限 |
|----------|-------------|-------------|
| 保守 | 5% | 15% |
| 均衡 | 10% | 25% |
| 进取 | 15% | 35% |
| 激进 | 25% | 50% |
