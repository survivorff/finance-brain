# 美林时钟 | Merrill Lynch Investment Clock

`🟡 进阶` `预计阅读：15 分钟`

> 核心问题：怎么根据宏观环境判断该买什么资产？

---

## 一句话总结

**美林时钟 = 用增长和通胀两个维度，把经济分成 4 个阶段，每个阶段对应不同的最优资产。简单但有用，是宏观投资的入门框架。**

---

## 各阶段详解

### 复苏期（Recovery）

```mermaid
graph TB
    A[特征] --> B[GDP 增速回升]
    A --> C[通胀仍低]
    A --> D[利率低位/正在见底]
    A --> E[失业率高位但开始下降]
    A --> F[企业盈利触底回升]
    
    G[最佳资产] --> H[股票（首选）]
    G --> I[偏价值/周期股]
    G --> J[小盘股弹性大]
    
    K[资产排序] --> L[股票 > 债券 > 大宗 > 现金]
```

**典型案例**：2009、2016、2020.4 后

### 过热期（Overheat）

```mermaid
graph TB
    A[特征] --> B[GDP 增速强劲]
    A --> C[通胀加速上升]
    A --> D[央行开始加息]
    A --> E[失业率低位]
    A --> F[资产价格泡沫]
    
    G[最佳资产] --> H[大宗商品（首选）]
    G --> I[原材料股]
    G --> J[抗通胀资产<br/>黄金/TIPS]
    
    K[资产排序] --> L[大宗 > 股票 > 现金 > 债券]
```

**典型案例**：2007、2021 后期、2008.5（油价 $147 时）

### 滞胀期（Stagflation）

```mermaid
graph TB
    A[特征] --> B[GDP 增速放缓]
    A --> C[通胀仍高位]
    A --> D[央行被迫维持紧缩]
    A --> E[企业盈利下滑]
    A --> F[最难处理的环境]
    
    G[最佳资产] --> H[现金（首选）]
    G --> I[黄金<br/>抗通胀+避险]
    G --> J[必需消费<br/>有定价权]
    
    K[资产排序] --> L[现金 ≈ 大宗 > 债券 > 股票]
```

**典型案例**：1970s 石油危机、2022 年

### 衰退期（Recession）

```mermaid
graph TB
    A[特征] --> B[GDP 负增长]
    A --> C[通胀回落]
    A --> D[央行开始降息]
    A --> E[失业率上升]
    A --> F[企业盈利恶化]
    
    G[最佳资产] --> H[长期债券（首选）]
    G --> I[防御股<br/>公用事业/医疗]
    G --> J[黄金可能受益]
    
    K[资产排序] --> L[债券 > 现金 > 股票 > 大宗]
```

**典型案例**：2008.10-2009.3、2020.3、1973-1975

## 美林时钟的局限

### 局限 1：央行干预扭曲

```mermaid
graph TB
    A[2008+ 央行大幅干预] --> B[QE/零利率]
    B --> C[改变了资产价格<br/>与基本面的关系]
    
    D[结果] --> E[股票/债券/黄金<br/>同时上涨]
    D --> F[传统美林时钟<br/>多次失效]
    
    style F fill:#e74c3c,color:#fff
```

### 局限 2：阶段没有清晰分界

```
现实中：
- 阶段之间是渐变的
- 有时跳过某个阶段
- 同一阶段持续时间差异大
- 不同国家可能处于不同阶段
```

### 局限 3：基于美国历史

```
美林时钟基于 1973-2004 美国数据。
中国/欧洲/新兴市场可能不完全适用。

中国特殊性：
- 政策周期 > 自然周期
- 房地产周期主导
- 货币传导机制不同
```

### 局限 4：未考虑流动性

```mermaid
graph TB
    A[美林时钟看的] --> B[增长 + 通胀]
    
    C[现实中还有] --> D[流动性环境]
    C --> E[政策导向]
    C --> F[市场情绪]
    
    G[启示] --> H[美林时钟<br/>是必要不充分]
```

## 实战使用美林时钟

### 步骤 1：判断当前阶段

```
1. 看 PMI（领先 3-6 个月）
2. 看 CPI/PCE（同步指标）
3. 看央行政策方向
4. 综合判断处于 4 个阶段中的哪个
```

### 步骤 2：对应资产倾斜

```
不是只买"最佳资产"，
而是在均衡配置基础上做倾斜（±10-20%）。

例：均衡配置 60/40
- 复苏期：调到 70/30（加股票）
- 过热期：调到 65/15/20（加大宗）
- 滞胀期：调到 30/30/40（多现金）
- 衰退期：调到 30/70（加长债）
```

### 步骤 3：监控信号

```mermaid
graph TB
    A[监控阶段转换] --> B[PMI 跌破 50<br/>= 扩张转衰退]
    A --> C[CPI 突破 4%<br/>= 过热信号]
    A --> D[央行开始降息<br/>= 衰退确认]
    A --> E[失业率上升 0.5%<br/>= 滞胀向衰退]
```

### 步骤 4：避免过度调整

```
太敏感 = 频繁交易 = 摩擦成本
太迟钝 = 错过最佳时机

平衡：每季度评估一次，半年/一年小幅调整
```

## 历史案例

### 2008-2010：完整周期

```
2008.5：过热顶部
- 油价 $147，通胀爆发
- 股市/商品高位
- 应该减仓股票，配置现金/黄金

2008.10：衰退底部确认
- 雷曼倒闭，金融危机
- 美联储紧急降息+QE
- 应该加仓长债

2009.3：复苏开始
- 股市触底
- 央行宽松
- 应该加仓股票

2010：扩张期
- 股市大幅上涨
- 持有股票 / 商品
```

### 2022：教科书式滞胀

```
2022.Q1：增长见顶
- 通胀飙升至 9%
- 美联储开始加息

2022.Q2-Q4：滞胀环境
- GDP 增速放缓
- 通胀仍高
- 股债双杀

最佳策略：
- 现金（涨息收益）
- 商品（俄乌战争+供应链）
- 黄金（避险+抗通胀）
- 避免：股票/债券
```
