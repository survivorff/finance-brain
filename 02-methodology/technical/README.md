## 技术分析的合理性与局限

```mermaid
graph TB
    A[支持] --> A1[价格反映所有信息]
    A --> A2[历史会重复（人性不变）]
    A --> A3[趋势会持续]
    
    B[反对] --> B1[随机游走理论]
    B --> B2[基本面才是决定因素]
    B --> B3[过度依赖会错过大机会]
    
    C[实用观点] --> C1[不要迷信，但也不要无视]
    C --> C2[作为辅助工具，不是核心]
    C --> C3[用于择时，不用于选股]
```

---

## 核心工具

| 工具 | 用途 |
|------|------|
| 趋势线 | 判断方向 |
| 移动平均线 (MA) | 平滑波动，识别趋势 |
| 支撑/阻力位 | 关键价格 |
| 成交量 | 确认趋势力度 |
| MACD | 趋势 + 动能 |
| RSI | 超买超卖 |
| 布林带 | 波动区间 |
