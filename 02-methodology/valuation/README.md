# 💎 估值方法 | Valuation

`🟡 进阶`

> 核心问题：一个资产到底值多少钱？

---

## 估值的三大方法

```mermaid
graph TB
    V[估值方法] --> A[绝对估值<br/>Intrinsic]
    V --> B[相对估值<br/>Relative]
    V --> C[资产估值<br/>Asset-Based]
    
    A --> A1[DCF 现金流折现]
    A --> A2[DDM 股利折现]
    
    B --> B1[P/E 市盈率]
    B --> B2[P/B 市净率]
    B --> B3[P/S 市销率]
    B --> B4[EV/EBITDA]
    B --> B5[PEG 增速调整]
    
    C --> C1[NAV 资产净值]
    C --> C2[清算价值]
```

---

## 各方法适用场景

| 方法 | 适用 | 不适用 |
|------|------|--------|
| DCF | 现金流稳定可预测 | 早期/亏损公司 |
| P/E | 盈利稳定的成熟公司 | 周期性公司、亏损公司 |
| P/B | 重资产公司（银行/地产） | 轻资产公司 |
| P/S | 营收稳定的公司、SaaS | 利润率差异大的行业 |
| EV/EBITDA | 跨国比较、高负债公司 | 资本结构差异大 |
| NAV | 房地产、控股公司 | 经营性公司 |
