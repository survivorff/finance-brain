## 新兴市场的"魔咒"

```mermaid
graph LR
    A[美元走强] --> B[新兴市场货币贬值]
    B --> C[美元债务负担加重]
    C --> D[资本外流]
    D --> E[本币更弱]
    E --> F[被迫加息保汇率]
    F --> G[经济下行]
    G --> H[股债双杀]
```

> 💡 这就是为什么美联储加息时，新兴市场往往会出现危机（亚洲金融危机、土耳其里拉、阿根廷比索...）。
