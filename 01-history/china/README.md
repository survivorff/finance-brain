        2022 : 三道红线影响显现
        2024-2025 : 政策转向，房地产寻底

## 几个核心问题

1. **中国为什么能在 40 年内从极贫到世界第二？**
   - 制度红利（市场化改革）
   - 人口红利（劳动力供给）
   - 全球化红利（加入 WTO）
   - 城市化红利（土地财政）
   - 后发优势（学习+追赶）

2. **当前面临的转型挑战是什么？**
   - 人口红利消失（2022 年人口负增长）
   - 全球化红利消失（去全球化）
   - 城市化红利消失（已 65%+）
   - 房地产红利消失（出清周期）

3. **未来的增长来源在哪里？**
   - 高端制造（新能源/半导体/AI）
   - 内需消费（仍在培育）
   - 服务业升级
   - 海外市场（一带一路）
