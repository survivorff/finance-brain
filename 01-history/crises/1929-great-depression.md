# 1929 大萧条 | The Great Depression

`🔴 高级` `预计阅读：25 分钟`

> 核心问题：1929 年股市崩盘怎么变成了 10 年大萧条？现在还有可能再来一次吗？

---

## 一句话总结

**大萧条不是因为股市崩盘，而是因为崩盘后政策极度错误。它定义了现代央行的所有"应该做什么"和"不应该做什么"。**

## 政策错误：为什么变成大萧条？

### 错误 1：美联储紧缩

```mermaid
graph TB
    A[1929 后美联储行为] --> B[反而紧缩货币<br/>1931 加息保金本位]
    
    C[当时金本位约束] --> D[英国脱离金本位<br/>资金涌入美国换金]
    D --> E[为防黄金外流<br/>美联储被迫加息]
    
    E --> F[加深通缩]
    F --> G[更多银行倒闭]
    
    style B fill:#e74c3c,color:#fff
```

弗里德曼后来论证：**大萧条本质是美联储造成的货币崩溃**。如果当时大胆放水，萧条不会这么严重。

### 错误 2：财政紧缩（胡佛政府）

```
1932 年胡佛：
- 平衡预算（紧缩支出）
- 加税
- 不愿大规模救济

→ 完全相反的方向
→ 加深萧条
```

### 错误 3：贸易保护

```mermaid
graph TB
    A[1930 斯姆特-霍利<br/>关税法案] --> B[关税提高到历史高位]
    B --> C[各国报复性关税]
    C --> D[全球贸易暴跌 65%]
    D --> E[全球萧条加深]
    
    style E fill:#e74c3c,color:#fff
```

> 💡 这就是为什么经济学家普遍反对贸易保护——大萧条期间各国"以邻为壑"，结果一起完蛋。

### 错误 4：金本位束缚

```
当时所有主要国家都在金本位下，
央行不能"印钱"救助。

最先脱离金本位的国家：
- 英国 1931（最早）
- 美国 1933
- 法国 1936（最晚）

→ 离开金本位的时间，几乎完全决定了
  各国走出萧条的速度
```


### 关键转折：脱离金本位

```mermaid

    B --> C[释放货币宽松空间]
```

### 1937 二次衰退：另一个教训

```
1937 年新政进展顺利，罗斯福政府决定：
- 平衡预算
- 美联储提高准备金率
- 太早紧缩

结果：1937-1938 经济再次衰退。

教训：恢复早期就紧缩 = 灾难性错误
现代版本：2010 年欧洲过早紧缩 → 欧债危机
```

## 大萧条会再来吗？

### 不太可能（短期）

```
现代央行有：
- 美联储独立性
- 各种工具（QE/财政协调）
- 经过 2008 检验的应对手册
- 全球协调机制

→ 不会出现 1929 那种"完全不作为"的情况
```

### 但风险在累积

```mermaid
graph TB
    A[当前隐忧] --> B[全球债务/GDP > 350%<br/>历史最高]
    A --> C[央行工具空间受限<br/>已经低利率+QE]
    A --> D[全球化倒退<br/>类似 1930s 关税战]
    A --> E[贫富分化<br/>类似 1920s]
    A --> F[地缘冲突<br/>1930s 也是地缘高风险期]
    
    G[结论] --> H[发生 1929 式萧条概率低<br/>但发生"长期低迷"的概率不低]
```

## 投资启示

### 大萧条期间什么资产保值？

```mermaid
graph TB
    A[1929-1933 表现] --> B[股票 -89%]
    A --> C[公司债 -50%]
    A --> D[国债 +10%（票息+避险）]
    A --> E[黄金 +75%（脱离金本位后）]
    A --> F[现金（如果银行没倒）+25%（通缩）]
    
    G[结论] --> H[现金/国债/黄金 是大萧条赢家]
```

### 长期视角

```
1929 顶部买入标普 500：
- 道指 1954 才回到 1929 高点（25 年）
- 但加上分红再投资，1944 年就回本了（15 年）

→ 即使在最差的时点买，长期持有+分红再投资
  仍然能赚钱（但需要极强的耐心）
```
