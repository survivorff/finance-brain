# 🇺🇸 美国经济史

> 理解美元、美联储、美国经济周期的演变，是理解全球经济的基础。

---

## 关键时间线

```mermaid
timeline
    title 美国经济 100 年（1929-2025）
    section 萧条与战后
        1929 : 大萧条开启
        1944 : 布雷顿森林会议（美元-黄金体系）
        1947 : 马歇尔计划
    section 黄金时代
        1950s-60s : 战后黄金期
        1971 : 尼克松关闭黄金窗口（金本位终结）
    section 滞胀与重生
        1973-1979 : 两次石油危机 + 滞胀
        1979-1987 : 沃尔克治通胀（暴力加息至 20%）
        1985 : 广场协议（美元贬值）
    section 大稳健时代
        1990s : 互联网繁荣
        2000-2002 : 互联网泡沫破裂
        2003-2007 : 房地产繁荣
    section 危机后
        2008 : 全球金融危机
        2009-2015 : 零利率 + QE 时代
        2015-2018 : 缓慢加息
        2020 : 疫情 + 史诗级宽松
        2022-2024 : 暴力加息治通胀
        2024+ : 缓慢降息
```

## 美联储的演变

| 时代 | 特征 | 代表主席 |
|------|------|----------|
| 早期（1913-1951） | 受财政部制约 | — |
| 独立时代（1951-1979） | 但通胀问题严重 | 多人 |
| 沃尔克时代（1979-1987） | 暴力加息治通胀 | Volcker |
| 大稳健（1987-2006） | 微调，市场最爱 | Greenspan |
| 危机时代（2006-2014） | QE 创新 | Bernanke |
| 正常化尝试（2014-2018） | 缓慢退出 | Yellen |
| 鲍威尔时代（2018-?） | 疫情 + 通胀 + 加息 | Powell |

## 美国经济的几个"长期叙事"

### 1. 美元霸权

```mermaid
graph TB
    A[美元霸权来源] --> B[二战后军事/经济实力]
    A --> C[布雷顿森林体系遗产]
    A --> D[石油美元（1974）]
    A --> E[美债深度+流动性]
    A --> F[盟友体系]
    
    G[挑战] --> H[去美元化趋势]
    G --> I[财政赤字]
    G --> J[政治极化]
```

### 2. 创新驱动

```mermaid
graph LR
    A[1970s 半导体] --> B[1990s 互联网]
    B --> C[2000s 移动互联网]
    C --> D[2010s 云计算/SaaS]
    D --> E[2020s AI]
```

每一轮技术革命都让美国保持竞争力。

### 3. 消费驱动

美国 GDP 中消费占比从 1950 年的 60% 上升到现在的 68%，**就业 → 收入 → 消费 → GDP** 形成闭环。
