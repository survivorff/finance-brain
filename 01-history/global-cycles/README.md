```mermaid
    subgraph "周期叠加"
```

| 库兹涅茨 | 15-25 年 | 建筑/城市化 | 房地产大周期 |
| 康波 | 40-60 年 | 技术革命 | 蒸汽 → 电力 → 信息 → AI？ |

## 康波周期（最长周期）

```mermaid
timeline

    section 第一次
        1782-1845 : 蒸汽机+纺织
    section 第二次
        1845-1892 : 钢铁+铁路
    section 第三次
        1892-1948 : 电气+化工
    section 第四次
        1948-1991 : 汽车+石化
    section 第五次
        1991-? : 信息技术
    section 第六次?
```
