# 📓 研究日志 | Research Journal

> 这是整个知识库中**长期最有价值的部分**。记录你的思考、判断、决策和复盘。

---

## 为什么要写日志？

```mermaid
graph LR
    A[记录判断] --> B[观察结果]
    B --> C[分析偏差]
    C --> D[修正认知]
    D --> E[更好的判断]
    E --> A
```

> "不记录的思考会消失。不复盘的决策不会进步。"

## 写作原则

1. **诚实** — 不美化自己的判断，错了就是错了
2. **具体** — 写清楚"为什么"，不只是"买了/卖了"
3. **及时** — 当天/当周记录，不要事后补（记忆会美化）
4. **结构化** — 用模板，方便回顾和统计
5. **定期回顾** — 每月/每季回看，找到自己的模式

## 复盘清单

每月问自己：
- [ ] 这个月做了几次决策？胜率多少？
- [ ] 最大的一次亏损是什么原因？
- [ ] 有没有"该做但没做"的操作？为什么？
- [ ] 我的认知有什么更新？
- [ ] 下个月最需要关注什么？

每季问自己：
- [ ] 我的投资体系有没有进化？
- [ ] 哪些判断框架被验证了？哪些需要修正？
- [ ] 我的情绪管理做得怎么样？
- [ ] 有没有在重复犯同一个错误？
